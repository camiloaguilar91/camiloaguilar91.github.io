Projects | Satellite Object Tracking

Summary

  • Object tracking is a challenging problem itself. The project will be focused on tracking dense population of moving objects in satellite videos. The main challenge of this application comes from the highly cluttered detection environment and the size of moving targets that can be very small. As a consequence, tracking objects based on their appearance is a challenge in itself. We therefore propose to work on a random finite set formulation of the tracking filter (PHD filter, see below) that formulates the tracking problem in a probabilistic problem where the main variable of interest is the intensity measure of a point process at the cost of losing individual targets. In a first step, we will focus on the insertion of a likelihood term based on a CNN output either in supervised learning (classification formulation) or in an unsupervised setting where variational autoencoders can indeed be used to map an input set of images onto a Gaussian distribution. A data likelihood term will therefore be built. This term will first be used in the classical PHD filter (SMC/ Gaussian or Box, to be defined during the study). In a second step, we will focus on the reformulation of the full PHD filter using CNNs by learning directly the different updating schemes (birth / prediction / correction) to be applied to a discrete representation of the intensity measure.
